Intro to HTML

Basic Terms

URL
the text string that defines a unique document, web address
Home page
beginning of a particular Web document or collection of documents
Web browser
a client program for navigating the web
  • Mozilla Firefox
  • Apple Safari
  • Microsoft Internet Explorer
  • Google Chrome
HTML
HyperText Markup Language, the language used to compose documents for Web delivery
HTTP
HyperText Transfer Protocol, the protocol used by Web servers to deliver requested documents to Web browsers

HTML: Elements

What goes between < and >

  • <p> Paragraph
  • <strong> Strong!
  • <div> Divisions of a document
  • <ul> <ol> <dl> Lists
  • <img /> Images
  • <br /> Breaks

Elements cannot be placed anywhere! How do we know where they can go? See the rules!

HTML: Attributes

What goes between <TAGNAME and >

  • href="" Links to another document
  • src="" Source of an external file
  • alt="" Alternate representation of an element
  • class="" Classify an element
<a class="home" href="http://.../">
    <img src="http://.../logo.png" alt="our logo" />
</a>

Not all attributes are available for every element! How do we know which are available? See the rules!

In-class Exercise

Build a simple webpage using basic html tags, such as:

<!doctype> <html> <head> <title> <body>

<h1> <h2> <p> <ul> or <ol> <li> <a> <img />

  • Use TextEdit or a similar basic text editor to create a plain text document.
  • Use the <a> tag to create both a link to another website and an “email me” link.
  • Find a random image online to use in the <img /> tag, being sure to include the width, height, alt attributes.
  • Create both an ordered list and unordered list to illustrate the difference in appearance in a browser.
  • Add an id attribute, with a value of your choosing, to denote a specific element/region of your page.
  • Create a link which points to that anchor using the href="#id" syntax.
  • Save the document with the .html extension.
