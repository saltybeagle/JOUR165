Intro to HTML

Basic Terms

URL
the text string that defines a unique document, web address
Home page
beginning of a particular Web document or collection of documents
HTML
HyperText Markup Language, the language used to compose documents for Web delivery
HTTP
HyperText Transfer Protocol, the protocol used by Web servers to deliver requested documents to Web browsers

HTML: Markup

Text that you mark-up

  • <h1>page heading</h1>
  • <p>paragraph of text</p>
  • <a href="http://google.com/">anchor/links</a>

HTML: What's Required

What does every HTML document need?

<!DOCTYPE HTML>
<html>
  <head>
    <title>sample title</title>
  </head>
  <body>
    Document body
  </body>
</html>

HTML: Elements

What goes between < and >

  • <p> Paragraph
  • <strong> Strong!
  • <div> Divisions of a document
  • <ul> <ol> <dl> Lists
  • <img /> Images
  • <br /> Breaks

Elements cannot be placed anywhere! How do we know where they can go? See the rules!

HTML: Attributes

What goes between <TAGNAME and >

  • href="" Links to another document
  • src="" Source of an external file
  • alt="" Alternate representation of an element
  • class="" Classify an element
<a class="home" href="http://.../">
    <img src="http://.../logo.png" alt="our logo" />
</a>

Not all attributes are available for every element! How do we know which are available? See the rules!
