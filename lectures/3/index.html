CSS

CSS Defined

Cascading stylesheets adjust the presentation of your markup

CSS: Referencing

  • Inline 
    <h1 style="font-size:35px;">Hello</h1>
  • Internal Stylesheet 
    <style type="text/css">
h1 {
  font-size:35px;
}
</style>
  • External Stylesheet 
    <link rel="stylesheet" type="text/css" href="styles.css" />

CSS Syntax

Selectors and properties…

selector [, selector2, ...][:pseudo-class] {
  property: value;
 [property2: value2;
  ...]
}
/* comment */

CSS: Selectors

  • Element Name
    • h1, p, div selects all the resective elements
  • Class value
    • .navigation selects <div class="navigation">...</div>
      • and <a class="navigation">...</a> and any other class="navigation"
  • ID value
    • #navigation selects <div id="navigation">...</div>

Properties

Far too many to list. A few categories of properties

  • Colors, borders, and images
    • border, background-color, (font-)color, background-image
  • Text
    • font-size, font-family, font-weight
  • Layout and position
    • height, width, margin, padding, position, left, right, top, bottom
  • See the CSS2 property reference, CSS3 current work
